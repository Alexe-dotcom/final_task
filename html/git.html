Git

  1. git init — это команда, которая превращает обычную папку на вашем компьютере в Git-репозиторий.
  2. git add — это команда в Git, которая добавляет содержимое рабочего каталога в область подготовленных файлов.
  3. git status - это основная команда Git, которая показывает текущее состояние рабочего каталога.
  4. git commit - это команда, которая создаёт новую версию кода в локальной истории.
  5. git push - это загрузка локального коммита в удаленный репозиторий (в основной).
  6. git merge [имя ветки] - это команда, которая вносит изменения указанной ветки в текущую ветку.
  7. git branch -d [имя ветки] - это команда, которая удаляет выбранную ветку.
  8. git branch [название ветки] - это команда, которая создаёт новую ветку.
  9. git checkout [название ветки] - это команда, которая осуществляет переход на нужную ветку (отслеживание Git).
